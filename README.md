ClothDrop Reborn lets you drape cloth over any object in seconds. No manual simulation setup, no keyframing. 
Select a plane and drop it, or draw one directly in the viewport.

## Installation

1. Download the latest release zip
2. In Blender go to **Edit → Preferences → Extensions → Install from Disk**
3. Select the zip and enable the addon

## Usage

Open the **N panel** in the 3D viewport and find the **ClothDrop Reborn** tab.

Draw your own rectangular plane above any object using **Draw Cloth**, or create your own plane and click **Create and Drop Cloth**

## Notes

- Sometimes the baking process takes a while, depending on your specs.
- For close-up hero shots, manual cloth simulation will always give better results. ClothDrop is built for speed, not perfection.  

## Requirements

- Blender 4.2.0 or newer
